코딩

웹페이지 템플릿을 코딩할 수 있는 기능

이 기능은 Javascrtipt를 사용하여 무엇이든 생성할 수 있도록 도와줍니다. 전역 변수 result에 문자열 html 데이터를 할당합니다. 리노는 이를 가져와 귀하의 웹페이지에 배치합니다. 리노 문서툴(Rino Docs)는 이 기능을 사용하여 사이드바에 페이지 링크 목록을 생성합니다.

data 와 props가 전역 변수로 이용 가능합니다. 구문은 {{()}}입니다

예시:

{{(
    result = `<ul class="menu-list">`
    let list = data.pageList;

    for(let i = 0; i < list.length; i++)
    {
        if(i == 0)
        {
            result = result + `
                <li><a href="/">${ list[i] }</a></li>
            `;
        }
        else
        {
            result = result + `
                <li><a href="/${ list[i] }.html">${ list[i] }</a></li>
            `;
        }
    }

    result = result + `</ul>`
)}}
