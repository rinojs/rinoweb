# Installation

You need the latest Node.js installed.

Then from terminal, enter the following:
```
npm create rino@latest
```
This will setup your project for building website.

Then you can start your development by:
```
npm run dev
```

You can also
```
npm i rinojs@latest
```
for manual use.
